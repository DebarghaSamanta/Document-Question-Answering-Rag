# PDF Document Question Answering System (RAG)

A Retrieval-Augmented Generation system for querying multiple PDF documents using:
- Hybrid retrieval (FAISS + BM25)
- Cross-encoder re-ranking
- OpenRouter LLM API
- Streamlit UI

## Status
Project initialized. Core pipeline under development.

## Planned Features
- Multi-document support  
- Hybrid semantic + keyword retrieval  
- Cross-encoder re-ranking  
- Answer with sources  
- Evaluation pipeline
